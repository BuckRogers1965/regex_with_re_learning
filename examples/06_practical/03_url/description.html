Extract HTTP and HTTPS URLs from text.

Pattern breakdown:

  • https?:// - HTTP or HTTPS protocol
  • (?:www\.)? - optional www prefix
  • [-a-zA-Z0-9@:%._\+~#=]{1,256} - domain characters
  • \.[a-zA-Z0-9()]{1,6} - TLD
  • (?:[-a-zA-Z0-9()@:%_\+.~#?&/=]*) - optional path and query
