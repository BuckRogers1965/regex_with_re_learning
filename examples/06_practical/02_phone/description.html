Match US phone numbers in various formats.

Pattern breakdown:

  • (?:\d{3}[-.]?)? - optional area code with optional separator
  • \d{3}[-.]? - 3 digits with optional separator
  • \d{4} - final 4 digits

This handles formats like 555-123-4567, 555.123.4567, 5551234567, and 123-4567.
